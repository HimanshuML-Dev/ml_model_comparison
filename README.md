# AI & Machine Learning Task 2
## Feature Engineering, Model Optimization & Performance Comparison

### Project Overview
This project focuses on building and evaluating machine learning models using the California Housing Dataset. The objective is to preprocess data, apply feature scaling, train multiple regression models, and compare their performance using standard evaluation metrics.

### Dataset
- Dataset: California Housing Dataset
- Source: Scikit-learn
- Target Variable: House Price

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

### Project Workflow

1. Import Required Libraries
2. Load California Housing Dataset
3. Data Exploration
4. Feature Engineering
5. Feature Scaling using StandardScaler
6. Train-Test Split
7. Model Training
   - Linear Regression
   - Ridge Regression
   - Decision Tree Regressor
8. Model Evaluation
   - RMSE (Root Mean Squared Error)
   - R² Score
9. Performance Comparison
10. Visualization of Actual vs Predicted Values

### Models Implemented

#### 1. Linear Regression
A basic regression algorithm that establishes a linear relationship between features and target values.

#### 2. Ridge Regression
An enhanced version of Linear Regression that uses L2 regularization to reduce overfitting.

#### 3. Decision Tree Regressor
A tree-based machine learning algorithm capable of learning complex non-linear relationships.

### Evaluation Metrics

#### RMSE (Root Mean Squared Error)
Measures prediction error. Lower RMSE indicates better performance.

#### R² Score
Measures how well the model explains the variance in the target variable. Higher R² indicates better performance.
